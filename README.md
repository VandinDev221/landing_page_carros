# Landing page — carros

Landing page em React (Vite) com Tailwind CSS e componentes de UI.

## Como executar

1. Instalar dependências: `npm install`
2. Ambiente de desenvolvimento: `npm run dev`
3. Build de produção: `npm run build`
4. Pré-visualizar o build: `npm run preview`

## Scripts

- `npm run lint` — ESLint
- `npm run lint:fix` — ESLint com correção automática
- `npm run typecheck` — verificação de tipos (jsconfig)
